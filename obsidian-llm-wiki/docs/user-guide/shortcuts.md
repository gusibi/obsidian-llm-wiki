# ⌨️ 快捷键与命令指南

## 🎯 如何触发插件功能

### 🚀 最快方法：命令面板
按 **Cmd/Ctrl + P**，然后输入 "claude" 或 "acp"，可以看到以下核心命令：

#### 核心功能
- **Open Claude Chat** (`open-claude-chat`) → 打开聊天界面
- **AI Edit Current File** (`ai-edit-current-file`) → AI 编辑当前文件
- **Quick Chat About Current Note** (`quick-chat-about-current-note`) → 针对当前笔记发起快速对话
- **Send Selection to Chat** (`send-selection-to-claude-chat`) → 将选中的文字发送到对话框

#### 快速操作
- **Open Wiki Panel** (`open-wiki-panel`) → 打开 Wiki 面板（目前等同于 Chat）

---

## ⚡ 设置快捷键（推荐）

为了提高效率，建议在 Obsidian 设置中为常用功能绑定快捷键：

1. **进入快捷键设置**：
   ```
   设置 (Settings) → 快捷键 (Hotkeys)
   ```

2. **搜索并设置**：
   搜索 "Claude ACP" 或 "LLM Wiki"，点击右侧的输入框并按下组合键。

### 🎨 推荐快捷键配置

| 功能 | 推荐快捷键 | 说明 |
|------|-------------|------|
| **打开聊天** | `Cmd + Shift + C` | 最常用，一键开启 AI 会话 |
| **AI 编辑** | `Cmd + Shift + E` | 对当前文件发起 AI 编辑指令 |
| **发送选区** | `Cmd + Shift + S` | 将当前笔记中的选区发送给 AI |
| **快速对话** | `Cmd + Alt + C` | 自动带上当前笔记的上下文 |
